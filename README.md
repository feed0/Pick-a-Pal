# Pick-a-Pal

Apple Developer Tutorial - [Create dynamic content](https://developer.apple.com/tutorials/develop-in-swift/create-dynamic-content#Use-a-text-field-to-enter-names)

## Tutorial
1. Write your `pals'` names
2. Hit the `remove picked name` toggle
3. `Pick` one pal at a time !!
